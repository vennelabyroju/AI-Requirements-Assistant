# AI Requirements Assistant

## Overview

This project demonstrates how Generative AI can support requirements gathering and documentation activities.

The prototype converts stakeholder meeting notes into:

- Functional Requirements
- User Stories
- Acceptance Criteria
- Traceability Matrices

## Business Problem

Many organisations capture requirements manually during workshops and stakeholder meetings.

This process can be time-consuming and may lead to inconsistent documentation and traceability gaps.

## Solution

The AI Requirements Assistant analyses meeting notes and generates structured project artefacts.
